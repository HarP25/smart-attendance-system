# Smart Attendance System

A web-based attendance management system built using **Flask** and **SQLite**.

## Features
- Add students
- Mark attendance by date
- View daily attendance
- Attendance summary with history
- Delete students safely

## Tech Stack
- Python
- Flask
- SQLite
- HTML

## How to Run
```bash
pip install flask
python app.py

Run the code once, experience the fun and experience.
